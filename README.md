# 影子贪吃蛇 Shadow Snake - Desktop Version

一个基于 **HTML + CSS + JavaScript Canvas** 制作的网页小游戏。
本版本为 **电脑端 / 键盘操作版本**，玩家需要控制贪吃蛇吃掉红宝石，同时躲避由自己过去路线形成的“影子障碍”。

游戏融合了传统贪吃蛇玩法、影子路线障碍、道具系统、外观进化和回溯机制，整体采用暗色霓虹风格设计。

## 游戏简介

《影子贪吃蛇》不是普通的贪吃蛇。
玩家每次吃到红宝石后，近期移动过的路线会先生成“影子预警”，随后凝结成实体影子障碍。玩家需要在越来越复杂的地图中规划路线，避免被过去的自己困住。

随着分数提升，游戏会逐步解锁更高速度、更高级外观、道具和影子回溯能力。

## 游戏特色

* **影子障碍机制**
  吃到红宝石后，玩家近期走过的路线会转化为影子障碍，增加路线规划难度。

* **预警与实体影子**
  影子会先以预警形态出现，随后变成可碰撞的实体障碍。

* **多阶段外观进化**
  达到指定分数后，蛇会解锁新的外观和速度等级，例如冰蓝核心、蓝绿电光、粉色霓虹、棱镜霓虹、黄金电弧等。

* **多种道具系统**
  游戏包含清影、超频、棱彩、磁铁、凝核等道具，不同道具有不同功能。

* **影子回溯机制**
  达到一定分数后可以获得回溯次数，在死亡时自动回到数秒前，给玩家一次补救机会。

* **霓虹视觉效果**
  使用 Canvas 绘制发光蛇身、影子障碍、粒子效果和浮动文字，整体视觉风格偏赛博霓虹。

## 进化机制

游戏会根据玩家分数逐步提升阶段。
分数越高，蛇的速度越快，外观越高级，同时影子障碍的生成节奏也会变化。

部分阶段会触发临时护盾和蛇身压缩，帮助玩家适应更高难度。

## 回溯机制

当玩家达到一定分数后，会获得“影子回溯”能力。
如果发生致命碰撞，系统会自动回退到几秒前，并清理周围部分影子，为玩家争取继续游戏的机会。

## 部署方式

本项目是纯静态网页，可以部署到 GitHub Pages。

部署步骤：

1. 创建一个新的 GitHub 仓库，例如：

```text
shadow-snake-desktop
```

2. 上传以下三个文件到仓库根目录：

```text
index.html
style.css
script.js
```

3. 进入仓库：

```text
Settings → Pages
```

4. 设置发布来源：

```text
Source: Deploy from a branch
Branch: main
Folder: /root
```

5. 点击保存，等待 GitHub Pages 生成网址。

生成后的链接通常类似：

```text
https://morikaaa.github.io/shadow-snake-desktop/
```
